Avant de bosser sur un fichier => Penser à pull
Après avoir bossé sur un fichier (Modification importante/conséquente effectuée ou tout simplement fini de bosser)
	=> Penser à pull, toujours
	=> Cliquer sur le cercle vide dans la fenêtre des modifications
	=> Cliquer sur "Stage all changes" dans la fenêtre qui apparaît
	=> Dans la section tout en bas "Commit Message", indiquer en "Summary" un court résumé des modifications effectuées
	=> Si besoin de plus de précisions, en ajouter dans "Description"
	=> Cliquer sur commit
	=> Enfin, pull
	=> Voilà !
